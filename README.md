# VueChat Pro

## 简介
基于 Vue 3 + Vite + Naive UI 的跨平台人工智能客户端。

## 核心特性
-  PWA 支持：零成本即可安装为桌面/移动端独立应用
-  极致轻量：借由此 PWA 加持体验原生快感
-  性能优异：集成打包体积分析，确保代码分块与按需加载

## Scripts (指令)

- 安装依赖: \pnpm i\ 
- 开发模式: \pnpm dev\ 
- 生产构建: \pnpm build\ 
- E2E测试: \pnpm test:e2e\ 
- 单元测试: \pnpm test\ 
- PWA离线缓存构建: 自动包含在 \uild\ 流程中
